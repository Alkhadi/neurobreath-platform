NeuroBreath Templates Bundle (Vector-ready + Vector-editable)

Folders:
- vector-ready/
  The original "vector-ready" pack (fully vector backgrounds; SVG + fields.json; includes some source rasters in place).
- vector-editable/
  The additional editable pack (template_edit_2/*) with SVG + fields.json and README.

Notes:
- SVG templates must not reference raster backgrounds. Use the provided validation script in your repo to enforce this.
- User image slots are <image id="slot_*" href=""> and should be filled at runtime with data:image/... or blob:... URLs.

Recommended repo location:
web/public/templates/nb-wallet/
