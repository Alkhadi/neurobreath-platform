## Usage Guidelines

- Use hero images for full-width hero sections
- SVG images are scalable and work well for backgrounds
- PNG images are best for illustrations and feature graphics
- WebP format provides better compression for web use
- Always include appropriate alt text when using images in HTML

## Image Paths

When referencing images from HTML files:
- From root HTML: `assets/img/filename.ext`
- From subdirectories: `../img/filename.ext` or `../../img/filename.ext` depending on depth

When referencing from CSS files:
- From `assets/css/`: `../img/filename.ext`
