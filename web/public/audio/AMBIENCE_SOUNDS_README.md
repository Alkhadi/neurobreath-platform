# Ambience Sound Files

The following ambient sound files are needed for the breathing techniques:

## Required Audio Files

| File Name | Description | Recommended Duration |
|-----------|-------------|---------------------|
| `cosmic-waves.mp3` | Space/cosmic ambient sounds | 2-5 minutes (looped) |
| `soft-rain.mp3` | Gentle rain sounds | 2-5 minutes (looped) |
| `morning-birds.mp3` | Birdsong and nature sounds | 2-5 minutes (looped) |
| `ocean-waves.mp3` | Ocean/sea wave sounds | 2-5 minutes (looped) |
| `forest-stream.mp3` | Forest with stream/water sounds | 2-5 minutes (looped) |
| `tibetan-bowls.mp3` | Tibetan singing bowl meditation | 2-5 minutes (looped) |
| `deep-meditation.mp3` | Deep meditation ambient music | 2-5 minutes (looped) |
| `spiritual-journey.mp3` | Spiritual/ethereal ambient sounds | 2-5 minutes (looped) |

## Audio Specifications

- **Format**: MP3 (recommended) or WAV
- **Sample Rate**: 44.1kHz
- **Bit Rate**: 128-192 kbps for MP3
- **Channels**: Stereo preferred, mono acceptable
- **Volume**: Normalized to prevent clipping
- **Loop-friendly**: Audio should loop seamlessly without clicks/pops

## Notes

- All audio files should be placed in this directory (`/public/audio/`)
- Files are loaded on-demand when user selects the ambience option
- Audio loops continuously during breathing sessions
- Volume is controlled by the user via the ambience volume slider
