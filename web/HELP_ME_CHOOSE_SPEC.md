# Help Me Choose Wizard (Spec)

## Purpose

Provide a short, respectful, non‑diagnostic wizard that helps users find a starter plan, tools to try now, and guides to explore. Educational only, privacy‑first, local‑only storage.

## Routes

- /uk/help-me-choose
- /us/help-me-choose
- /uk/help-me-choose/results
- /us/help-me-choose/results

## Wizard Flow

- 5 short steps with multi‑select where appropriate
- Skippable options and “Not sure” choices
- Audience selector early
- Support needs, environment, time preference, optional profile

## Results

- Recommended starter journey
- 2–4 tools to try now
- 3–6 guides to learn more
- Trust & safety links
- Local‑only storage

## Guardrails

- No diagnosis or medical advice
- Internal links only
- Privacy‑first, local‑only storage
- UK/US emergency guidance only in safety panel

## Accessibility

- Keyboard navigable
- Focus management between steps
- Clear button labels

## SEO

- Indexable wizard page with rich explanatory content
- Results page noindex and excluded from sitemap
