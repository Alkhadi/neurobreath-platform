# About Authority Stack

This document summarises the About authority stack implemented in the Next.js App Router (/web).

## Routes (localised)

- /uk/about
- /us/about
- /uk/about/mission
- /us/about/mission
- /uk/about/methodology
- /us/about/methodology
- /uk/about/who-its-for
- /us/about/who-its-for
- /uk/about/use-responsibly
- /us/about/use-responsibly
- /uk/about/how-we-update
- /us/about/how-we-update
- /uk/about/faq
- /us/about/faq
- /uk/about/language
- /us/about/language

## Design principles

- Educational information only; no medical claims.
- Clear, calm, professional tone.
- UK/US localisation for spelling and terminology.
- Trust and safety links present on all pages.

## Validation

- Use `npm run about:validate` to confirm H1, metadata, internal links, and Trust Centre presence.
